Danger Zone

Irreversible actions — read carefully before proceeding

⚠
Destructive Actions

Delete all backup files

Permanently removes all .sql files and MongoDB BSON folders from disk. Connection history entries are also cleared. Your connections and schedules are kept.

Reset everything

Deletes all connections, schedules, backup history, and every backup file on disk. ChronoBase returns to a completely blank state. There is no undo.
